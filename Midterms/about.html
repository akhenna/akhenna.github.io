ABOUT THE PROJECT

DRAGON BALL API

A fan-made REST API inspired by the legendary Dragon Ball universe. Explore characters, transformations, planets, and iconic moments from the series.

What is this?

The Dragon Ball API is a comprehensive REST API inspired by the iconic television series Dragon Ball.

This API grants users access to an extensive database featuring hundreds of characters, images, transformations, and planets from the Dragon Ball universe.

It includes canonical information from:

  • Dragon Ball Z
  • Dragon Ball GT
  • Dragon Ball Super
  • Dragon Ball Movies
  • Dragon Ball Heroes

Who are you?

“I'm Antonio Alvarez, a developer who loves creating things and has a deep passion for Dragon Ball.”

He studied at Relevant in the Polo de Contenidos Digitales in Malaga and created this project to help students, anime fans, and developers worldwide.

Why did you build this?

The project started from a passion for Dragon Ball and the desire to help fellow students.

As a student himself, Antonio understood the challenges developers face while learning APIs and building projects.

This API combines anime, technology, and education into one interactive experience for the community.

Technical Stuff

The Dragon Ball API is developed using modern technologies:

  • NestJS + Express Engine
  • TypeScript Framework
  • MySQL Database
  • Cloudinary API
  • AWS EC2 Hosting
  • GitHub Actions CI/CD
  • React + Vite
  • Material UI
  • Netlify Deployment

This setup provides scalability, smooth deployment, and an excellent user experience.
